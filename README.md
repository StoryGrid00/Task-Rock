# Task Rock - Firebase Push Notifications Edition

A fun task management app with Jerry the Rock, your virtual pet companion, now featuring Firebase push notifications that work even when the app is closed!

## 🔥 **New Firebase Features**

### **Dual Notification System**
- **🔔 Local Notifications**: Browser-based notifications when tasks are due
- **🔥 Firebase Push Notifications**: Cloud-based notifications that work when app is closed

### **Cross-Platform Support**
- **Desktop**: Chrome, Firefox, Safari, Edge
- **Mobile**: iOS Safari, Chrome Mobile, Firefox Mobile
- **PWA**: Add to home screen for enhanced experience

### **iOS Compatibility**
- **Full Firebase support** when added to home screen
- **Automatic detection** of iOS devices
- **User guidance** for optimal setup

## 🚀 **Quick Start**

1. **Open** Task Rock in your browser
2. **Navigate to Settings** (⚙️ Settings section)
3. **Enable notifications**:
   - Toggle **Local Notifications** for browser-based alerts
   - Toggle **Firebase Push Notifications** for cloud-based alerts
4. **Grant permissions** when prompted
5. **Create tasks** with due dates
6. **Receive notifications** even when app is closed!

## 📱 **iOS Setup Instructions**

For the best Firebase notification experience on iOS:

1. **Open** Task Rock in Safari
2. **Tap the Share button** (square with arrow)
3. **Select "Add to Home Screen"**
4. **Open** Task Rock from your home screen
5. **Enable Firebase notifications** in Settings
6. **Enjoy** push notifications that work when app is closed!
