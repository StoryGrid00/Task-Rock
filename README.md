# 🗿 Task Rock - UI Improved Version

A gamified task management app with Jerry the Rock companion, featuring enhanced background notifications and improved user interface.

## 🎯 **Latest Updates - UI Improvements**

### ✅ **What's New**
- **Removed Firebase Toggle**: Simplified notification settings by removing confusing Firebase option
- **Enhanced Spacing**: Professional padding and alignment throughout notification settings
- **Cleaner Status Indicator**: Single "Background Notifications" status with improved visual design
- **Better Typography**: Improved font sizes and readability
- **Professional Layout**: Enhanced visual hierarchy and user experience

## 🚀 **Features**

### **🔔 Background Notifications**
- **True Background Operation**: Notifications work even when app is closed
- **Progressive Reminders**: 20min, 10min, 5min, 2min, and due-time alerts
- **Action Buttons**: Complete, Snooze, View directly from notifications
- **Cross-Platform**: Works on all devices and browsers
- **No Server Required**: Completely client-side operation

### **🎮 Gamification System**
- **Jerry the Rock**: Your animated companion that grows with your productivity
- **Health System**: Stay motivated with Jerry's health that decreases with overdue tasks
- **Experience Points**: Earn XP and level up Jerry through task completion
- **Achievement System**: Unlock achievements for various milestones
- **Streak Tracking**: Build and maintain daily completion streaks
- **Rock Shop**: Customize Jerry with hats, accessories, and paint jobs

### **📝 Task Management**
- **Smart Prioritization**: Easy, Medium, Hard difficulty levels with point values
- **Due Date Tracking**: Set specific dates and times for tasks
- **Priority Levels**: Low, Medium, High priority classification
- **Progress Tracking**: Comprehensive statistics and completion rates
- **Daily Challenges**: Special tasks with bonus rewards

## 🛠️ **Technical Stack**

- **Frontend**: Pure HTML5, CSS3, JavaScript (ES6+)
- **Storage**: IndexedDB for persistent local data
- **Notifications**: Web Notifications API + Service Workers
- **PWA Ready**: Manifest and service worker included
- **No Dependencies**: Zero external libraries or frameworks

## 📦 **Installation & Deployment**

### **GitHub Pages Deployment**
1. Fork or download this repository
2. Enable GitHub Pages in repository settings
3. Select source as "Deploy from a branch" → main branch
4. Your app will be available at `https://yourusername.github.io/repository-name`

### **Local Development**
```bash
# Clone the repository
git clone https://github.com/yourusername/task-rock-ui-improved.git

# Navigate to directory
cd task-rock-ui-improved

# Serve locally (Python 3)
python -m http.server 8000

# Or use any local server
# Visit http://localhost:8000
```

### **Custom Domain Setup**
1. Add a `CNAME` file with your domain name
2. Configure DNS settings to point to GitHub Pages
3. Enable HTTPS in repository settings

## 🔧 **Configuration**

### **Notification Settings**
- Notifications are enabled by default
- Users can toggle notifications in Settings
- Background notifications work automatically when enabled
- No additional setup required

### **Customization**
- Modify colors in CSS variables at the top of `index.html`
- Adjust notification timing in `task-rock-background-notifications.js`
- Add new achievements in the achievements array
- Customize Jerry's shop items in the shop section
